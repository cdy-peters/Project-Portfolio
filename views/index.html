Project - Recommendations for Spotify


Technologies Used

Python, Flask, Spotify API


Project Purpose

Recommendations is a web app that was built to explore the Spotify API whilst also being my first full stack project. Development evolved into exploring sessions to provide a dynamic experience for users followed by a brief introduction to threading.


Project Description

Recommendations provides accurate song recommendations given a selected playlist, ordering recommendations by similarity to the theoretical average song in the playlist. The average song is based on the characteristics of each song in the playlist as provided by the Spotify API. Using cosine similarity, a comparison is made between the average song and each recommendation's characteristics to return a value from 0 to 1, representing how similar the songs are.
